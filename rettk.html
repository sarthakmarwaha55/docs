Responsive Ecologies toolkit

Responsive Ecologies Tangible toolkit (RET/tk) aims to bring rapid prototyping and programming to unexperienced designers and developers. It is currently under development with the end goal of providing an authoring toolkit that allows web developers to create cross-device applications.

You can find more about the project here: RE/tk project

I work on the design and development of the platform. Tools used:

  • Node.js
  • JavaScript
  • HTML & CSS
  • Adobe XD
